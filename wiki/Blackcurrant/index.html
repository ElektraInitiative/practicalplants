Blackcurrant


is a common name used for a number of distinct species.

Plants with the common name Blackcurrant

R

  • Ribes americanum (American Blackcurrant)
  • Ribes lacustre (Prickly Blackcurrant)

R cont.

  • Ribes nigrum (Blackcurrant)
  • Ribes petiolare (Wetern Blackcurrant)

S

  • Salvia microphylla (Blackcurrant Sage)
