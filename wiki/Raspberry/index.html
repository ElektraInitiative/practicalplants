Raspberry


is a common name used for a number of distinct species.

Plants with the common name Raspberry

R

  • Rubus acaulis (Dwarf Raspberry)
  • Rubus calycinus (Wild Raspberry)
  • Rubus deliciosus (Rocky Mountain Raspberry)
  • Rubus ellipticus (Golden Evergreen Raspberry)
  • Rubus idaeus (Raspberry)

R cont.

  • Rubus illecebrosus (Strawberry-Raspberry)
  • Rubus leucodermis (Whitebark Raspberry)
  • Rubus nepalensis (Nepalese Raspberry)
  • Rubus niveus (Ceylon Raspberry)
  • Rubus occidentalis (Black Raspberry)

R cont.

  • Rubus parvifolius (Japanese Raspberry)
  • Rubus pedatus (Trailing Wild Raspberry)
  • Rubus rosaefolius (Mauritius Raspberry)
  • Rubus strigosus (American Red Raspberry)
