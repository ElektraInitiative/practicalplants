is a common name used for a number of distinct species.

Plants with the common name Hawthorn

A

  • Acer crataegifolium (Hawthorn-Leaved Maple)
  • Aponogeton distachyos (Water Hawthorn)

C

  • Crataegus apiifolia (Parsley-Leaved Hawthorn)
  • Crataegus calpodendron (Pear Hawthorn)
  • Crataegus chrysocarpa (Fireberry Hawthorn)
  • Crataegus coccinoides (Kansas Hawthorn)
  • Crataegus columbiana (Columbian Hawthorn)

C cont.

  • Crataegus douglasii (Black Hawthorn)
  • Crataegus laevigata (Midland Hawthorn)
  • Crataegus macrosperma (Big-Fruit Hawthorn)
  • Crataegus monogna (Hawthorn)
  • Crataegus monogyna (Hawthorn)
  • Crataegus nigra (Hungarian Hawthorn)
  • Crataegus pruinosa (Frosted Hawthorn)

C cont.

  • Crataegus punctata (Dotted Hawthorn)
  • Crataegus reverchonii (Reverchon's Hawthorn)
  • Crataegus rivularis (River Hawthorn)
  • Crataegus submollis (Quebec Hawthorn)

R

  • Rhaphiolepis indica (Indian Hawthorn)
  • Rhaphiolepis umbellata (Japanese Hawthorn)
