Fig


is a common name used for a number of distinct species.

Plants with the common name Fig

C

  • Carpobrotus acinaciformis (Hottentot Fig)
  • Carpobrotus aequilaterus (Sea Fig)
  • Carpobrotus deliciosus (Sweet Hottentot Fig)
  • Carpobrotus edulis (Hottentot Fig)

C cont.

  • Chenopodium ficifolium (Fig-Leaved Goosefoot)

F

  • Ficus carica (Fig)
  • Ficus coronata (Sandpaper Fig)
  • Ficus macrophylla (Moreton Bay Fig)

F cont.

  • Ficus palmata (Wild Fig)

O

  • Opuntia phaeacantha (Bastard Fig)

S

  • Scrophularia buergeriana (Buerger's Figwort)
  • Scrophularia nodosa (Knotted Figwort)
