Melon


is a common name used for a number of distinct species.

Plants with the common name Melon

C

  • Citrullus lanatus (Water Melon)
  • Cucumis melo (Melon)
  • Cucumis melo agrestis (Wild Melon)

C cont.

  • Cucumis melo cantalupensis (Cantaloupe Melon)
  • Cucumis melo chito (Orange Melon)
  • Cucumis melo conomon (Pickling Melon)

C cont.

  • Cucumis melo flexuosus (Serpent Melon)
  • Cucumis melo inodorus (Honeydew Melon)
  • Cucumis melo momordica (Snap Melon)
