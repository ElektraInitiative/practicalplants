Tomato


is a common name used for a number of distinct species.

Plants with the common name Tomato

C

  • Cyphomandra betacea (Tree Tomato)

P

  • Physalis pruinosa (Strawberry Tomato)

S

  • Solanum aethiopicum (Mock Tomato)
  • Solanum lycopersicum (Tomato)

S cont.

  • Solanum pimpinellifolium (Currant Tomato)
  • Solanum sisymbriifolium (Sticky Nightshade, Litchi Tomato, Morelle De Balbis)
